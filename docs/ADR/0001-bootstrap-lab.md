# ADR 0001: Bootstrap del laboratorio Moltbot

## Estado
Aceptada

## Contexto
Se necesita iniciar un laboratorio nuevo para Moltbot upstream con foco en reproducibilidad, seguridad y trazabilidad.

## Decisión
Crear estructura mínima de repo, plantillas de configuración y scripts `verify_*` desde H0.

## Consecuencias
- Base consistente para avanzar por hitos H1-H5.
- Aún faltan integración upstream y proveedor real.
