# cunningham - Moltbot Upstream Lab

Laboratorio reproducible para instalar y ejecutar **Moltbot upstream**, conectarlo con proveedores externos por API, cambiar de modelo por perfiles e integrar extensiones de comunidad con trazabilidad y foco en seguridad.

## Alcance
- Repo autónomo (no depende de Lucy-C).
- Framework y tooling open-source.
- Secretos fuera de git.
- Integraciones externas con pin por commit/tag y revisión mínima.

## Estado
- H1 en implementación: instalación upstream vanilla de OpenClaw/Moltbot.

## Quickstart
```bash
./scripts/verify_smoke.sh
./scripts/verify_upstream.sh
```
